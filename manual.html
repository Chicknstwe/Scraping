How to install
Install with XAMPP
First of all, download XAMPP here and install it.
Look for the directory where XAMPP has been installed and open it. There should be a folder called "htdocs". Open it and extract here every file from Scraping tool.rar (or Scraping tool.tar.gz).
Now we have to edit a couple of things in a file called php.ini. You may find it at "Xampp/php/". Now find:
  • "max_execution_time" and set its value to 9999999999. If there are semicolons (";") before "max_execution_time", remove them.
  • "output_buffering" and set it to On or any integer number if it is off. If there are semicolons (";") before "output_buffering", remove them.
Install Pear, OLE and Excel packages
Excel exporter needs the following packages to be installed. You may find detailed info to install them here:
  • Link to Pear package page.
  • Link to OLE package page.
  • Link to Excel package page.
Twitter dev tokens
Tweets collector require some tokens povided by Twitter. To get yours, you must sing up at apps.twitter.com and "Create a new app". You may use the following data to fill in some of the fields:
  • Name: Scraping tool.
  • Website: https://github.com/Chicknstwe/Scraping.
After that, just go to the "Keys and Access Tokens" section to get your own tokens and copy&paste them into their respective fields in "Settings" section of this application.
Install curl cert
Tweets collector uses curl so you have to download this cretificate and place it somewhere in Xmapp's folder, for example at "Xampp\php\extras\openssl\".
Now go to "php.ini", the same file we edited previously. Find "curl.cainfo" and "openssl.cafile" and set them to the ABSOLUTE PATH of the "cacert.pem" certificate, wherever you placed it.


Warnings and recommendations for use
  • Scraping tool execution time will be longer depending on whether you run it with more or less executions. It will be definitely longer if you make it download images and documents.
  • NEVER stop the scripts (closing the tab or the browser, for example) while they are running. If you do, it will produce many errors on registers. You can fix those errors by using the reset buttons at config section, but you will lose all the information.
  • The scraping toool search is case sensitive, so make sure you use the correct keywords.
  • The scraping tool use REGEX to look for keywords in html files. They are powerful, but has some limitations, so use this tool carefully.
  • You should run this code in an "up to date" browser.
  • This code is open source and has a MIT license that covers everything except sorttable.js and TwitterAPIExchange.php libraries. You can use it without restrictions but don't claim it as yours.
