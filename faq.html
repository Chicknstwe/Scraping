FAQ
I'm just getting started, how do I use this script?
First of all, you should go to Manual section and read everything carefully.
This app uses websites, keywords and twitters accounts to run the scripts, so you should add the websites/twitter accounts and keywords at "Configuration/Settings" section.
Now, you can use "Scraping tool" to search for words in one or more websites and save the results(websites, images, docs and other files) or Tweets collector to get some tweets.
To see the results, go to "Results explorer" section.
I just reset one of the registers. Can I recover the info?
No.

Is there a way to delete my twitter dev credentials?
Just leave empty the fields at "Configuration/Settings" and click "Apply".
What type of files can I download if I check the documents checkbox when I run the scraping script?
Here's the list: ".pdf", ".doc", "docx", ".xls", ".odt", ".ods", ".rtf", ".dot", "dotx", ".txt", ".xml", "docm", "dotm", ".dic", ".rar", ".zip", "r.gz", ".tar", ".bz2".
